About This Project

Project Overview

The Bitcoin Z-Score Analyzer is a research-oriented trading system designed to analyze Bitcoin price behavior, generate rule-based trading signals, and evaluate performance using financial risk metrics. It integrates real market data, statistical modelling, automated alerts, and a user-friendly backtesting interface.
